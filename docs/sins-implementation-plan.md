# sins酸性ストレートページ実装方針

`app/features/sins.feature` を満たすための実装方針。

---

## セクション順序（変更後）

```
1. メインビジュアル        ← 説明 + 施術例トップ写真（現在は写真なし）
2. 対象となる悩み          ← 現在の「こんな方におすすめです」を移動
3. 詳しい説明              ← 現在の「当店の特徴」
4. 注意事項                ← 現在の位置から移動
5. ショートの施術例        ← 新規追加（写真1枚）
6. ミディアムの施術例      ← 新規追加（before/after 4枚）
7. ロングの施術例          ← 新規追加（before/after 4枚）
8. 料金                    ← 現在の位置から移動
9. 追加料金について        ← 新規追加
```

現在の順序：メインビジュアル → 悩み → 特徴 → 注意事項 → 料金

---

## 差分

| 仕様 | 現状 | 対応 |
|------|------|------|
| 説明 + 施術例がページ先頭 | 写真なし・テキストのみ | 施術例トップ写真を追加 |
| 詳しい説明が悩みの後 | 悩みの前に特徴がある | 順序を入れ替え |
| ショート・ミディアム・ロングの施術例 | なし | 新規追加 |
| 追加料金についての説明 | なし | 新規追加 |
