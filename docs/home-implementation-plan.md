# ホームページ実装方針

`app/features/home.feature` を満たすための実装方針。

---

## セクション順序

```
1. コンセプト               ← 最初のコンテンツ（ヒーロー削除）
2. 外観・内観写真
3. 酸性ストレート紹介 + 予約ボタン
4. 酸性ストレート詳細       ← before/after写真含む
5. メニュー
6. FAQ
7. 予約
8. アクセス
```

---

## 各セクションの実装方針

### コンセプト

- ヒーローセクションを削除し、コンセプトをページ先頭に配置する
- 以下が伝わるコピーにする：
  - 素朴で昔ながらの雰囲気（飾らない、地域密着）
  - 長く通いながらケアをしていく関係性
  - 技術・知識へのこだわり

### 外観・内観写真

- `exterior.avif` / `interior.avif` を横並びで表示
- キャプションなし

### 酸性ストレート紹介 + 予約ボタン

- 詳細セクションより前に置く導入的な紹介
- 予約ボタンをセット

### 酸性ストレート詳細

- `sins-before-after.avif` を表示
- 説明・料金・リンク
